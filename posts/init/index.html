Project objective

Main goal of this project is to learn some ARM platfrom exploiting basics, which will further lead to develop some skills in Apple’s devices exploiting methods.

Tools and devices

Device I already had and could use to this project is iPhone 4S and iPhone 5. Why these ones? They have ARMv7 processors which means they have 32-bit architecture - seems a little bit easier for the begining to me. These devices are also relatively cheap (hello from year 2020), and further on I’m aiming in developing skills in exploiting and debugging Apple devices. If you want to follow this guide step-by-step you should have MacBook computer.

Prepare device

Jailbreaking

If you have previous experience with jailbreaking or if you are iOS developer, jailbreaking iPhone 5 with iOS 10.3.4 is quite easy. Steps I took to jailbreak this one are:

  1. I assume you are on MacBook and you have AppleID created. Open Xcode and start simple project for iOS.
  2. Connect your iPhone to Mac, in Xcode select iPhone device as build target and try to run this project on it. If you have build problems - read them carefully and try to fix them. Simple project you can run will be added here in sources //TODO
  3. If your simple project run on device means we have connection established, your signing certificate is working, we’re good to go for jailbreaking.
  4. Download H3lix.ipa from https://h3lix.tihmstar.net/ (for older iOS versions: PhÅ“nix.ipa from https://phoenixpwn.com/)
  5. Download AppSigner from https://dantheman827.github.io/ios-app-signer/
  6. Follow the instructions to re-sign app.
  7. Open Xcode, form the topbar choose Window -> Devices and Simulators
  8. Choose your iPhone device, click (+) to add your re-signed Phoenix.ipa to install it on device
  9. Now on your device open Phoenix app, click jailbreak and keep fingers crossed ðŸ˜‰
  10. In few minutes you should have succefully jailbroken iPhone 4S.

Same steps as above are suitable to jailbreak other older devices (just the matter of finding proper for iOS system version jailbreak tool).

What to install?

Will start with simple tools: clang, lldb or gdb. Also add some tools from Cydia:

  • OpenSSH
  • Substrate Safe Mode
  • wget, curl, rsync
  • theos (https://github.com/theos/theos/wiki/Installation-iOS)

How to connect to device - SSH

After jailbreaking and installing OpenSSH on device, you should be abble to connect to it. Find your device IP address (in device WiFi network settings) and ssh into it:

ssh root@192.168.1.45

Default password is alpine.
